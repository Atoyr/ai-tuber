# ゲーム知識: サンプルゲーム

これはペルソナパッケージの知識ファイルの**実例**(生きた契約ドキュメント)。
`knowledge/<名前>.md` に 1ゲーム = 1ファイルで置き、実況起動時に
`--game <名前>` / 環境変数 `GAME_KNOWLEDGE` / Studio の「ゲーム知識」欄で選択すると、
この内容がシステムプロンプトに結合される(詳細は docs/persona-architecture.md)。

## どんなゲームか
- ジャンル: 見下ろし型のアクションパズル
- 目的: 各ステージの出口まで、ブロックを押したり壊したりしながらたどり着く

## 実況で使える用語
- コアブロック: 壊すとステージ全体のギミックが動く青いブロック
- スター: 各ステージに3つ隠されている収集要素

## 実況で触れると良いポイント
- 残りスター数や画面上部のタイマーの変化
- コアブロックを壊す前の「どう動くかわからないドキドキ」
- 失敗してもリトライが早いので、失敗は明るくネタにする
